# Decision Log

## Current Decisions

- iCloud is the primary working storage.
- iCloud Keychain is the default secret store.
- GitHub private repo stores only code and structure.
- NAS is backup-only.
- The Mac mini is the reference machine for the workspace layout.

## Reasoning

- iCloud keeps the workspace available across devices.
- Apple keychain tools avoid introducing a separate password manager.
- GitHub versioning is useful for code and folder structure, not for private data.
- NAS is best used as a recovery layer, not as the main working copy.
